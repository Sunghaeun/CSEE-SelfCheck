# CSEE-SelfCheck : 공학프로젝트기획 수업 수강 확인 서비스

## 소개

한동대학교 학생들이 공학프로젝트 수업을 수강하는데에 있어, 본인이 수강을 할 수 있는지 가능 여부를 셀프 체크하고, 어떤 학생들이 수강 할 수 있는지 여부 등을 확인하는 서비스입니다.
